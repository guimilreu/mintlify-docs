## Base URL

Todos os endpoints usam a base:

```
https://api.socialsell.ai/v1
```

## Autenticação

Todas as requisições exigem um Bearer token no header `Authorization`:

```http
Authorization: Bearer sk_live_...
```

Cada API Key está vinculada a uma organização e possui scopes específicos. Obtenha sua chave no painel da SocialSell em **Configurações → Integrações → API Keys**.

## Paginação

Os endpoints de listagem usam paginação por **cursor**:

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `limit` | integer | Itens por página (padrão: 25, máx: 100) |
| `cursor` | string | Token retornado em `next_cursor` da resposta anterior |

## Rate Limits

- **Geral**: 60 requisições/minuto por API Key
- **Envio de mensagens**: 30 msgs/min por instância/página

Respostas `429 Too Many Requests` incluem o header `Retry-After`.

## Planos

Alguns recursos exigem o plano **Growth+**:
- Broadcasts (disparos em massa)
- AI Agents

Requisições a esses endpoints com plano Starter retornam `403 Forbidden`.
